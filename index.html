Beach Resort

Beach Resort is a lodging and accomodations web application built with React. It employs plain CSS and styled components for the styling and is fully responsive. Basic functionalities include sorting rooms by different features such as price, breakfast and other filters. React Router is used for navigating between pages.

Mobile Store

Mobile Store is an e-commerce web application built with React. It is fully responsive and implements PayPal Sandbox. Basic functionalities include adding items to cart, removing items from cart, increaing item counts and making payment with PayPal payment gateway. React Router is used for navigating between pages.

Visit site  

Fashion Store

Fashion Store is a desktop e-commerce web application built with React. It uses firebase authentication for creating shopping accounts and signing in. It implements Stripe Payment Gateway. Basic functionalities include adding items to cart, removing items from cart and increaing item counts. React Router is used for navigating between pages.
